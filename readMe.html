Below is the algorithm used for developing a Password for the User


  1. Reverse the Website Name
  2. Ask the user to pick a letter to remove from the Website Name
  3. Ask the user to replace the letter they just removed with a different letter
  4. Find out what the last letter of the website name is
  5. Replace the last letter of the Website Name with the next letter in the alpahbet Ex : The letter 'a' becomes 'b'
  6. Now set the Ascii Value to the Value of the previous letter. Using the example above. The Ascii Value would be 97, because that's the Ascii Value of A in Java
  7. Finally, Print out the Password!!
